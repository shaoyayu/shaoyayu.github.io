技术栈

​ 目前的技术栈方向，主要是微服务 Spring Cloud Alibaba与 大数据工程。相比微服务我更喜欢大数据工程，喜欢分布式计算和分布式系统搭建。融汇贯通的使用大数据技术和微服务技术同时应用开发是我的主要学习方向。目前主要学习DDD领域驱动设计和微服务中的组件,同时还在学习大数据的CDH技术。

​ 继往开来，在以后的职业规划中，主要在GIS服务系统和数字孪生方向。希望有志同道合的朋友一起学习。

​ 大学本科专业是计算机科学与技术(移动软件开发)，涉及的课程包括，C语言、Java语言、操作系统、计算机网络、数据库、数据结构与算法、Java Web系统、Android应用开发、大数据原理Hadoop、高等数学、离散数学、线性代数和概率论等学科。

​ 除学习以外的知识，主要学习包括高并发网络编程、 Spring、Spring MVC、MyBatis、Spring Boot等基础的java web开发技术，在大数据方向主要包括 HDFS、MapReduce、Hive、HBase、Zookeeper等其他组件（主要目前使用于数仓建设）。

​ 主要使用的数据存储包括：MySQL、MongoDB、Redis、ES等。

​ 负载均衡技术主要了解LVS和熟悉Nginx技术。

​ 微服务技术主要学习使用的技术栈是Spring Cloud Alibaba。
